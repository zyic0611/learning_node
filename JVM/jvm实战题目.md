# 实战题目



## cpu 飙升100% 怎么救火

排查4步骤

1. top命令 发现PID 123【进程号 为xxx的CPU的占用率很高
2. top -Hp 123  -H看线程 -p指定进程号 发现这个java进程 有一个线程id为 456的吃光了CPU
3. printf "%x\n" 456  把线程号转换为16进制 因为jm打出来的底层日子 线程号全部都是16进制 为101e
4. jstack 123 | grep -A 20 101e 导出123进程的堆栈 搜索101e这个线程  打印他的前后20行日志
5. 结果会出现此线程正在执行的java代码位置 打开对应位置排查
